# Quick Start Guide

## 🚀 Starting the Demo

The demo is currently running but needs to be restarted to pick up all the new files.

### Restart the Dev Server

1. **Stop the current server:**
   - In your PowerShell terminal, press `Ctrl + C`
   
2. **Start it again:**
   ```bash
   npm run dev
   ```

3. **Open in browser:**
   - Go to: http://localhost:5173/

4. **Login:**
   - Email: `manager@mkopa.com` (or any email)
   - Password: `password` (or any password)

## 🔧 Troubleshooting

### Issue: "does not provide an export named..."
**Solution:** Restart the dev server (Ctrl+C, then `npm run dev`)

### Issue: Blank page or errors
**Solution:** 
1. Check browser console (F12)
2. Clear browser cache (Ctrl+Shift+Delete)
3. Restart dev server

### Issue: Changes not showing
**Solution:** Hard refresh the browser (Ctrl+F5)
